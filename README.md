# DNO 修改器 A1.0 Stable

这是《Diplomacy is Not an Option》的单机修改器。
对应版本：dno 1.0.82 r
适用场景：单机、离线、自用娱乐。

- 如果 Windows 或杀毒软件提示未知程序，这是因为工具没有代码签名。请确保你的文件是直接从 GitHub Releases 发布页下载：https://github.com/XXLBABY/Diplomacy_is_Not_an_Option_Trainer/releases

## 需要准备什么

1. Windows 系统。
2. 已安装《Diplomacy is Not an Option》dno 1.0.82 r
3. 游戏目录里已经安装 BepInEx 5。

如果你不知道有没有安装 BepInEx：

- 打开游戏目录。
- 如果能看到 `BepInEx` 文件夹，通常说明已经安装。
- 如果没有，需要先安装 BepInEx 5，并至少启动一次游戏。

## 推荐安装方法

最简单的方法：

1. 完全退出游戏。
2. 运行：

```text
DNO-Trainer-A1.0-Stable-Installer.exe
```

3. 如果安装器没有自动找到游戏目录，点击浏览，选择游戏根目录。

游戏根目录指的是能看到游戏 exe 和 `Diplomacy is Not an Option_Data` 文件夹的位置。

4. 点击安装。
5. 安装完成后启动游戏。
6. 进入一局游戏后，运行安装目录里的：

```text
DinaoExternalTrainer.exe
```

7. 在修改器里打开总开关，或按小键盘 `.` 打开总开关。

## 手动安装方法

如果安装器不能使用，也可以手动安装：

1. 完全退出游戏。
2. 解压本压缩包。
3. 把解压后的 `DNO-Trainer-A1.0-Stable` 文件夹放到游戏根目录里。
4. 双击运行：

```text
安装助手.bat
```

5. 看到安装完成后，启动游戏。
6. 进入一局游戏后，再运行：

```text
DinaoExternalTrainer.exe
```

7. 在修改器里打开总开关，或按小键盘 `.` 打开总开关。

## 如果安装助手失败

通常是因为没有把文件夹放在游戏目录里。

你可以这样做：

1. 把 `DNO-Trainer-A1.0-Stable` 文件夹移动到游戏根目录。
2. 再运行 `安装助手.bat`。

如果仍然失败，可以手动安装：

1. 打开游戏目录。
2. 打开 `BepInEx\plugins` 文件夹。
3. 把本文件夹里的 `DinaoStorageAutoMarker.dll` 复制进去。
4. 重启游戏。

## 启动顺序

推荐顺序：

1. 启动游戏。
2. 进入一局游戏。
3. 启动 `DinaoExternalTrainer.exe`。
4. 打开修改器总开关。
5. 使用小键盘热键。

## 热键

所有热键都是小键盘。

| 热键 | 功能 |
| --- | --- |
| 小键盘 `.` | 总开关 |
| 小键盘 `1` | 设置木材 |
| 小键盘 `2` | 设置石材 |
| 小键盘 `3` | 设置铁 |
| 小键盘 `4` | 设置食物 |
| 小键盘 `5` | 设置灵魂水晶 |
| 小键盘 `6` | 设置金币 |
| 小键盘 `7` | 建造 / 升级立即完成 |
| 小键盘 `8` | 研究立即完成 |
| 小键盘 `9` | 训练立即完成 |
| 小键盘 `/` | 清除召唤修改 |
| 小键盘 `*` | 应用 / 取消召唤修改 |

## 功能说明

### 资源修改

可以修改：

- 木材
- 石材
- 铁
- 食物
- 灵魂水晶
- 金币

在修改器窗口里输入数值，然后按对应热键或点击 `Set`。

### 立即完成

可以开启：

- 建造 / 升级立即完成
- 研究立即完成
- 训练立即完成

这些功能是开关式的，再按一次可以关闭。

### 召唤修改

可以选择要召唤的单位，设置数量和持续时间。

- 数量：每种单位召唤多少个。
- 持续时间：单位存在多少秒。
- 持续时间填 `0`：永久召唤。

设置完成后按小键盘 `*` 应用。

想恢复原版召唤，按小键盘 `/` 清除。

## 已知限制

A1.0 Stable 是稳定使用版，不是所有实验功能都包含。
A1.0 Stable 只完成修改资源、快速建造/研究/训练、使用基础召唤功能(友方单位)。
后续版本主要开发/修复 敌对召唤单位 超级单位。

已知限制：

- 敌对单位的完整控制、移动攻击、巡逻等不作为本版重点。
- 超级单位的子召唤物阵营问题没有处理。
- 敌对单位血条颜色没有处理。

## 卸载

完全退出游戏后，运行：

```text
禁用助手.bat
```

或者手动删除：

```text
游戏目录\BepInEx\plugins\DinaoStorageAutoMarker.dll
```

## 常见问题

### 按热键没反应

请检查：

- 修改器总开关是否打开。
- 是否使用小键盘，不是主键盘数字键。
- 游戏是否已经进入局内。
- helper 是否已经安装到 `BepInEx\plugins`。

### 资源修改失败

请先在游戏里触发一次资源变化，例如建造、消耗或获得资源，再尝试修改。

### 游戏启动后没有加载 helper

检查：

- 是否安装了 BepInEx 5。
- `DinaoStorageAutoMarker.dll` 是否在 `BepInEx\plugins` 文件夹里。
- 游戏是否完全重启过。

### 修改器窗口太碍事

可以使用窗口里的置顶和夜间模式选项，按自己习惯调整。
